# Phase 2b：施测 Mini-IPIP（人格量表）

## 前置准备

读取 `doc/mini-ipip-scale.md` 获取完整题目和计分键（注意反向计分题）。

## 量表说明（告知用户）

```
接下来是「大五人格量表」（Mini-IPIP），共 20 题，分 2 批作答，约 5-8 分钟。
请用 1-5 分作答：
1 = 非常不符合   3 = 中立/不确定   5 = 非常符合
请描述你「通常的状态」，而非理想状态或近期特殊情况。
```

## 施测流程（分 2 批）

### 第 1 批（题目 1-10）

| 题号 | 题目 |
|:---:|:---|
| 1 | 我是聚会中的焦点人物。 |

| 3 | 我会立即完成杂务。 |

| 5 | 我有生动的想象力。 |

| 8 | 我经常忘记把东西放回原处。 |
| 9 | 我大部分时间都很放松。 |
| 10 | 我对抽象概念不感兴趣。 |

### 第 2 批（题目 11-20）

| 题号 | 题目 |
|:---:|:---|
| 11 | 在聚会上我会与许多不同的人交谈。 |

| 18 | 我把事情搞得一团糟。 |
| 19 | 我很少感到忧郁。 |
| 20 | 我没有很好的想象力。 |

## 计分（重要：反向计分处理）

**反向计分题**（第 6、7、8、9、10、15、16、17、18、19、20 题）：
- 转换公式：计入分 = 6 - 原始分
- 即：1→5, 2→4, 3→3, 4→2, 5→1

**维度计算**（使用计入分后的分数）：

| 维度 | 题号（R=反向） | 计算 |
|:---|:---|:---|
| 外向性 | 1, 6R, 11, 16R | 4题平均 |
| 宜人性 | 2, 7R, 12, 17R | 4题平均 |
| 尽责性 | 3, 8R, 13, 18R | 4题平均 |
| 神经质 | 4, 9R, 14, 19R | 4题平均 |
| 开放性/智力 | 5, 10R, 15R, 20R | 4题平均 |

**展示计算过程**：逐维度展示原始分→反向处理→均分，让用户可以核验。

## 水平描述参考

| 维度均分 | 水平描述 |
|:---:|:---|
| 4.5-5.0 | 极高 |
| 3.5-4.4 | 偏高 |
| 2.5-3.4 | 中等 |
| 1.5-2.4 | 偏低 |
| 1.0-1.4 | 极低 |

## 完成后操作

1. 将 5 个维度的均分和水平描述写入画像文件 `## 五、人格` 字段
2. 删除该字段的 `（AI推断）` 标注（如有）
3. 将 `采集阶段` 更新为包含 `mini_ipip_done`
4. 检查：若 AMS、RCSS、Mini-IPIP 均已完成，将采集阶段更新为 `all_scales_done`
5. 给出人格特征的简短解读（2-3句，聚焦最突出的 1-2 个维度）
6. 引导用户：

```
人格量表完成！所有量表均已填写完毕。
你可以：
- 输入「查看画像」查看完整画像并进行审核确认
- 输入「生成综合解读」让 AI 撰写综合分析
```
